📝 Answer Guide

Use this guide if you're having trouble with the questions:

Question 1: Use the definition of derivative to find f'(x) for f(x) = 2√x

Step 1: Find f(x+h) = 2√(x+h)

Step 2: Set up the difference quotient: [f(x+h) - f(x)]/h = [2√(x+h) - 2√x]/h

Step 3: Rationalize and simplify:

Multiply by [√(x+h) + √x]/[√(x+h) + √x]:

[2√(x+h) - 2√x]/h × [√(x+h) + √x]/[√(x+h) + √x] = [2(x+h) - 2x]/[h(√(x+h) + √x)] = [2h]/[h(√(x+h) + √x)] = 2/[√(x+h) + √x]

Final answer: As h→0, f'(x) = 1/√x

Question 2: Match each function with its correct derivative

Correct matches:

  • f(x) = √(3x - 12) → f'(x) = 3/(2√(3x - 12))
  • f(x) = √(x² + 25) → f'(x) = x/√(x² + 25)
  • f(x) = 1/(3x - 2) → f'(x) = -3/(3x - 2)²
  • f(x) = x²/(x + 3) → f'(x) = (x² + 6x)/(x + 3)²
Question 3: Secant line as h approaches 0

Answer: "The secant line becomes the tangent line as h approaches 0"

Note: Your answer must include both "tangent" and either "becomes" or "approaches"

Question 4: Step-by-Step for f(x) = √(5x - 1)

Step 1: f(x+h) = √(5(x+h) - 1) = √(5x + 5h - 1)

Step 2: [f(x+h) - f(x)]/h = [√(5x + 5h - 1) - √(5x - 1)]/h

Step 3: Rationalize: [√(5x + 5h - 1) - √(5x - 1)]/h × [√(5x + 5h - 1) + √(5x - 1)]/[√(5x + 5h - 1) + √(5x - 1)]

= [(5x + 5h - 1) - (5x - 1)]/[h(√(5x + 5h - 1) + √(5x - 1))] = [5h]/[h(√(5x + 5h - 1) + √(5x - 1))] = 5/[√(5x + 5h - 1) + √(5x - 1)]

Step 4: As h→0: f'(x) = 5/(2√(5x - 1))

Question 5: Derivative of a constant function f(x) = 7

Answer: f'(x) = 0

Note: The derivative of any constant function is always zero
