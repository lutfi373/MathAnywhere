2.9 Linear Approximations & Differentials

This section covers two closely related concepts that are fundamental in calculus and have extensive practical applications in science and engineering.

🎯 Learning Objectives

  • Understand the concept of linear approximation
  • Apply the linear approximation formula
  • Calculate and interpret differentials
  • Solve real-world estimation problems
  • Analyze approximation accuracy

🔗 Prerequisites

  • Understanding of derivatives
  • Knowledge of the tangent line
  • Basic function evaluation
  • Chain rule (for complex functions)

2.9.1 Linear Approximations

Linear approximation uses the tangent line at a point to estimate nearby function values. This technique is particularly useful when exact calculations are difficult or when we need quick estimates.

Linear Approximation Formula:

$f(x_0 + \delta x) = f(x_0) + f'(x_0) \delta x$


Where:

$\delta x = x - x_0$ (change in x)

$x_0$ = Initial value / Estimated value

$x$ = Latest value / Given value

📝 Key Points

  • The approximation is most accurate when $|\delta x|$ is small
  • Choose $x_0$ to be a value where $f(x_0)$ and $f'(x_0)$ are easy to compute
  • The linear approximation is the equation of the tangent line at $(x_0, f(x_0))$
  • Error increases as we move further from the approximation point

2.9.2 Differentials

Differentials provide a way to express small changes in functions and are closely related to linear approximations.

Differential Formula:

$df = f'(x_0) \delta x$


Where:

$\delta x = x - x_0$ (change in x)

$x_0$ = Initial value / Estimated value

$x$ = Latest value / Given value

🔄 Relationship to Linear Approximation

The differential $df$ represents the approximate change in the function value:

$f(x_0 + \delta x) \approx f(x_0) + df$

This shows that differentials are the "change" part of linear approximation.
