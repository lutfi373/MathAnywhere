Question 1: When do you use the Product Rule for differentiation?

Answer: "When you have two functions being multiplied together" or "When differentiating a product of functions"

Example: For f(x) = (2x + 1)(x¬≤ - 3), use Product Rule: f'(x) = (2)(x¬≤ - 3) + (2x + 1)(2x)

Formula: If f(x) = u(x) ¬∑ v(x), then f'(x) = u'(x)v(x) + u(x)v'(x)

Question 2: Which function requires the Quotient Rule?

Answer: B) f(x) = (x¬≤ + 3x - 2)/(x¬≤ - 1)
