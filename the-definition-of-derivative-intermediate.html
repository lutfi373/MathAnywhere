📝 Answer Guide

Use this guide if you're having trouble with the questions:

Question 1: Formal definition of derivative

Answer: f'(x) = lim(h→0) [f(x+h) - f(x)]/h

This is the formal limit definition that must include the limit notation.

Question 2: Derivative of f(x) = x³

Step 1: Set up: lim(h→0) [(x+h)³ - x³]/h

Step 2: After expanding and simplifying: f'(x) = 3x²

Question 3: Non-differentiable point of |x|

Point: x = 0

Explanation: The left and right limits differ at x = 0, creating a corner/cusp

Question 4: Left and right derivatives of |x| at x = 0

Left derivative: -1 (slope going left)

Right derivative: 1 (slope going right)

Question 5: Derivative of √x

Setup: lim(h→0) [√(x+h) - √x]/h

Result: After rationalizing: f'(x) = 1/(2√x)

Question 6: Differentiability implies continuity

Answer: If a function is differentiable at a point, it must be continuous there

Note: The converse is not always true - continuity doesn't guarantee differentiability

Question 7: Piecewise function differentiability

Answer: a = 2

For differentiability at x = 1, the left derivative (2x at x=1 gives 2) must equal the right derivative (a)
