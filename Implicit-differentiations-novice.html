Use this guide if you're having trouble with the questions:

Question 1: What is implicit differentiation used for?

Answer: "Implicit differentiation is used when we cannot easily solve y in terms of x."

Other acceptable words: isolate, express, find

Explanation: When equations like x¬≤ + y¬≤ = 25 cannot be easily rearranged to y = f(x), we use implicit differentiation.

Question 2: Chain rule in implicit differentiation

Correct Answer: A) Chain rule: d/dx(y¬≤) = 2y ¬∑ dy/dx

Explanation: Since y depends on x, we use the chain rule: outer derivative √ó inner derivative.

Question 3: Circle equation x¬≤ + y¬≤ = 25

Answer: dy/dx = -x/y

Step-by-step:

  • Differentiate: 2x + 2y(dy/dx) = 0
  • Solve for dy/dx: 2y(dy/dx) = -2x
  • Final: dy/dx = -x/y
Question 4: Product rule xy = 12

Answer: dy/dx = -y/x

Step-by-step:

  • Product rule: d/dx(xy) = x(dy/dx) + y(1) = 0
  • Simplify: x(dy/dx) + y = 0
  • Solve: x(dy/dx) = -y
  • Final: dy/dx = -y/x
Question 5: Slope at point (3, 4)

Answer: -3/4

Step-by-step:

  • From Question 3: dy/dx = -x/y
  • Substitute (3, 4): dy/dx = -3/4
  • The slope at point (3, 4) is -3/4
