📝 Answer Guide

Use this guide if you're having trouble with the questions:

Question 1: Purpose of linear approximation

Answer: "Linear approximation is used to estimate function values near a known point."

Other acceptable words: approximate, find, calculate, predict

Explanation: When we can't easily compute exact values, we use linear approximation to get close estimates.

Question 2: Linear approximation formula

Correct Answer: A) L(x) = f(a) + f'(a)(x - a)

Explanation: This is the tangent line equation rearranged: y = f(a) + f'(a)(x - a)

Question 3: Linear approximation of √x at a = 9

Answer: L(x) = 3 + (1/6)(x - 9)

Step-by-step:

  • f(9) = √9 = 3
  • f'(x) = 1/(2√x), so f'(9) = 1/(2√9) = 1/6
  • L(x) = f(9) + f'(9)(x - 9) = 3 + (1/6)(x - 9)
Question 4: Estimate √8.8

Answer: 2.967 (approximately)

Step-by-step:

  • L(8.8) = 3 + (1/6)(8.8 - 9)
  • L(8.8) = 3 + (1/6)(-0.2)
  • L(8.8) = 3 - 0.2/6 = 3 - 0.033... ≈ 2.967
Question 5: Find dy when y = x²

Answer: 0.6

Step-by-step:

  • f'(x) = 2x, so f'(3) = 2(3) = 6
  • dy = f'(3) × dx = 6 × 0.1 = 0.6
